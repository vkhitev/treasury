# Treasury | Казначейство

## Курсова робота з дисципліни "Організація баз даних та знань"

### Як запустити сайт?
1. Завантажити проект
2. $ cd . && npm install
3. $ npm start

### Що знаходиться в папках?
- app_api : моделі, рути та контролери для API проекту
- app_server : рути та контролери для веб-застосування
- bin : скрипт для завантаження серверу
- db_queries : деякі скрипти для бази даних
- public : клієнтська частина веб-застосування
- sample_data : скрипти, що генерують дані для вставки в бд
